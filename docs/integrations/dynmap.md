# dynmap Integration

dynmap integration renders faction territory overlays on the web map.

## Config Keys

- `integrations.dynmap`

## Setup

1. Install dynmap.
2. Set `integrations.dynmap: true`.
3. Restart and open your dynmap web UI.

## Verify

1. Confirm faction territories are visible in dynmap.
2. Claim/unclaim a chunk and verify the map updates.
