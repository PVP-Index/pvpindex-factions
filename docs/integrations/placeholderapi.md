# PlaceholderAPI Integration

PlaceholderAPI exposes PvPIndex Factions values for scoreboards, chat formats, and other plugins.

## Config Keys

- `integrations.placeholderapi`

## Setup

1. Install PlaceholderAPI.
2. Set `integrations.placeholderapi: true` in `config.yml`.
3. Restart the server.

The expansion is registered automatically when PlaceholderAPI is present.
