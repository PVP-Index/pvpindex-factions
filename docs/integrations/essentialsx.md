# EssentialsX Integration

When enabled, `/f home` teleport routing can use EssentialsX.

## Config Keys

- `integrations.essentialsx.enabled`

## Setup

1. Install EssentialsX.
2. Set `integrations.essentialsx.enabled: true`.
3. Restart and test `/f home`.

## Verify

1. Teleport to faction home.
2. Confirm behavior aligns with EssentialsX teleport handling.
