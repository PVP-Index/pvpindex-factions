# EzEconomy Integration

EzEconomy is supported as an economy provider through Vault.

## Setup

1. Install Vault.
2. Install EzEconomy.
3. Keep `integrations.vault: true`.
4. Restart and check that Vault resolves EzEconomy as the active economy provider.

## Verify

1. Execute economy-backed faction actions (create, claim, bank).
2. Confirm balances and transactions are reflected in EzEconomy.
