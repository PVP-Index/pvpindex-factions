# Configuration

Configuration is split into focused files:

- `config.yml` for gameplay, economy, power, claim behavior and integrations
- `database.yml` for storage backend and DB connection settings
- `messages.yml` for localization/translations
- `gui.yml` for `/f` GUI structure and actions

Use `/fa reload` after text/config edits where supported; restart when in doubt.
