# Homes and Warps

Home:

- Set: `/f sethome`
- Use: `/f home`
- Remove: `/f unsethome confirm`

Warps:

- Teleport: `/f warp <name>`
- Set/update: `/f warp set <name> [here|x y z world]`
- Delete: `/f warp delete <name>`
- List: `/f warp list [page]`

Config keys:

- `factions.max-warps`
- `factions.warp.list.page-size`
