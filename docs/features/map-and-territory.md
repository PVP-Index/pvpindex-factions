# Map and Territory Display

Commands:

- `/f map once`
- `/f map on`
- `/f map off`

Features:

- Interactive map cells with hover and click actions
- Wilderness click-to-claim (`/f claim at ...`)
- Territory transition notifications with faction info hover

Config keys:

- `factions.map.once-radius`

Per-player control:

- `/f notify territory on|off`
