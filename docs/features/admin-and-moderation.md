# Admin and Moderation

Admin root:

- `/fa`

Core actions:

- `/fa bypass`
- `/fa claim`
- `/fa unclaim`
- `/fa disband <faction>`
- `/fa reload`

Permission:

- `factions.admin`

Operational note:

- Use admin claim/unclaim carefully in production to avoid player claim conflicts.
