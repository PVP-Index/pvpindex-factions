# Factions GUI

Commands:

- Open default: `/f` (no args) or `/f gui`
- Open specific menu: `/f gui <menu>`

Capabilities:

- Config-driven menus/items/actions
- Dynamic placeholders for faction and player stats
- Click actions mapped to `/f` command flows

Config source:

- `gui.yml`
