# Faction Fly

Command:

- `/f fly`

Behavior controls:

- Enable/disable feature globally
- Disable on threat
- Restrict to own territory

Config keys:

- `factions.fly.enabled`
- `factions.fly.disable-on-threat`
- `factions.fly.require-own-territory`
