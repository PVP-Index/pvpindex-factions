# Permissions

Primary admin permission:

- `factions.admin`

Command-level permissions follow:

- `factions.cmd.*`

Examples:

- `factions.cmd.claim`
- `factions.cmd.unclaim`
- `factions.cmd.notify`
- `factions.cmd.bank.history`

Reference source of truth: `src/main/resources/plugin.yml`.
