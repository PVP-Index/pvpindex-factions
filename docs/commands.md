# Commands

## Player root

- `/f` (aliases: `/faction`, `/factions`)

## Admin root

- `/fa` (aliases: `/factionadmin`)

## Discoverability

- `/f help` gives guided onboarding and grouped command sections.
- `/fa help` lists administration commands.
- `/f top [power|bank|land]` supports leaderboard sorting.
